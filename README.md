# ITEC5025 Week 1 Assignment - Basic Python Program

## Project Description
This repository contains the coursework for the Week 1 assignment of ITEC5025. The main objective is to initialize the fundamental structure for a chatbot application using Python.

The script demonstrates:
- Basic output functionality
- Graceful error handling using `try-except` blocks
- Modular design principles

## Files
- `chatbot.py`: The main Python script that initializes the chatbot and prints a "Hello, World!" message with formatting.

## How to Run
To execute the program, verify you have Python installed and run the following command in your terminal:

```bash
python chatbot.py
```

## Expected Output
```
------------------------------
SYSTEM STATUS: ONLINE
OUTPUT: Hello, World!
------------------------------
```
